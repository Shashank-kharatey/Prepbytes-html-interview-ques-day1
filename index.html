Q.1 Difference between div and span?

Ans. "Div" tag defins a "Devision" or "Section" in HTML. It acts as a container element that is required to hold a specific type of contents or data in our webpage. It is an inline block level element i.e it acquires the space properties according to the data it contains.
Whereas,
"Span" tag is purely an inline element in HTML. It usually be used for containing the small portion of content which needs to be edit instantly or at short notice, basically it needs not for us to define a class or id for "span" rather we can use directly attributes in it.

Q.2 Difference between class and IDs?

Ans. "ID" or Identifier is an attribute used in HTML in order to provide an unique location/mark/name/identifier to the element, so that it can be called for modifications in future.
Whereas
"Class" is defined in regards to provide any range of properties collectivily to the element bearing the same class. It can be assigned to any no. of elements that are supposed to depict the same qualities.

Q.3 What are self closing tag?

Ans. self closing tags sometimes also known as 'empty tags/void tags/singelton tags' etc, these tags have no data content to present and hence do not have any ending part in their structure rather they use right shift slash in their origin tag to end. for eg. img tag, hr tag

Q.4 How many types of Headings are in HTML?how to decide which to use where.

Ans. There are six levels of Headings defined in HTML. They ranges from H1 to H6 with H1 being the highest font size and H6 with the least. It depends on the category of the content that what type of Heading tag would be favourable.
Heading tags are block level by nature.

Q.5 Difference between block and inline?

Ans. Block level elements always starts on a new line, and the browsers automatically add some space (a margin) at top and bottom of the element. Block level elements always takes upto the full width available on the web page. eg. paragraph element and div element.
Inline elements does not start on a new line. It only takes up as much widht as necessary. eg. span element

Q.6 Difference between em and strong element?

Ans. The "Strong" element should be used to indicate strong importance/seriousness/urgency, like to indicate key phrases in a text for someone skimming it. It Bolds up the given content.
The "em" element or "Emphasize" element used to represent stress emphasis,like when you'd read the emphasized text in a different tone of voice. It present its content in italic form

Q.7 What are attributes?

Ans. "Attributes" are the language defined keywords that depicts the specific function of an element they used within. We can say that attributes show the additional functionality of an element.
